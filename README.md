# Gabinet Weterynaryjny Futrzak

Responsywna, statyczna strona internetowa przygotowana do publikacji w GitHub Pages.

## Pliki

- `index.html` – struktura i treść strony
- `styles.css` – cały wygląd i wersja mobilna
- `script.js` – menu mobilne, animacje oraz automatyczny status otwarcia
- `assets/gabinet-futrzak-tablica.jpg` – zdjęcie gabinetu

## Publikacja na GitHub Pages

1. Utwórz nowe repozytorium na GitHubie.
2. Wgraj całą zawartość tego folderu do głównego katalogu repozytorium.
3. Wejdź w `Settings` → `Pages`.
4. W sekcji `Build and deployment` wybierz:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
5. Zapisz ustawienia i poczekaj kilka minut.

## Co warto uzupełnić przed publikacją

- Potwierdzić pełny zakres usług.
- Potwierdzić, czy sobotnie godziny obowiązują w każdą sobotę.
- Dodać własne zdjęcie lek. wet. Beaty Pędziak-Koperskiej.
- Dodać politykę prywatności, jeśli później pojawi się formularz lub analityka.
- Podmienić skrócony link Google Maps na docelowy link do wizytówki, jeśli będzie dostępny.

## Edycja danych

Najważniejsze dane, takie jak telefon, adres, godziny i opinie, znajdują się w pliku `index.html`.
Kolory i wygląd można zmienić na początku pliku `styles.css` w sekcji `:root`.
